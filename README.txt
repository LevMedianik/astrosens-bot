AstroSens — LLM-based Telegram-бот, способный:
1. Общаться на тему космоса, экзопланет и зарождения жизни
2. Обрабатывать и анализировать статьи в формате PDF, DOCX, TXT
3. Отвечать на вопросы на основе загруженных файлов
4. Выполнять краткий пересказ содержимого файла

---

Используемые инструменты и навыки:
- Язык программирования		Python
- Telegram-бот			python-telegram-bot
- Генерация ответов		OpenRouter API, LLM модель deepseek/deepseek-chat-v3-0324:free
- Работа с документами		PyMuPDF (fitz), python-docx
- Обработка текста		LangChain, FAISS, sentence-transformers
- Векторные представления	HuggingFaceEmbeddings
- Хранилище окружения		.env, python-dotenv
- Дополнительно			re, uuid, requests, os, asyncio

---

Возможности бота
/start		Приветствие и краткое описание возможностей
/askfile	Задать вопрос по загруженному файлу (PDF, DOCX, TXT)
/summary	Краткий пересказ содержания последнего файла
/reset		Очистка текущего контекста файла для отправки нового
/help		Справка

---

Как запустить?
Так как бот уже развёрнут на сервере, следует лишь запустить по ссылке t.me/AstroSens_bot
